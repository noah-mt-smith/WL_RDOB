# WL_RDOB

R code for rank dependent optimal behaviour (switching and waiting) experiments

Note: The analysis file uses cleaned RDS files for data. If the analysis file isn't running, you must first run the data cleaning file. Running the data cleaning file will generate all RDS files used in the analysis.

| File                                            | GitHub path                 |
|-------------------------------------------------|-----------------------------|
| Read Me                                         | WL_RDOB/README.md           |
| Experiment 1 (switching) raw data               | WL_RDOB/WL_switching_df.csv |
| Experiment 2 (waiting) raw data                 | WL_RDOB/WL_waiting_df.csv   |
| Mood (experiments 1 + 2) raw data               | WL_RDOB/WL_mood_df.csv      |
| Data cleaning                                   | WL_RDOB/data_cleaning.R     |
| Data analyses                                   | WL_RDOB/data_analyses.Rmd   |
| Electronic supplementary material (in progress) | WL_RDOB/supp_info.pdf       |
